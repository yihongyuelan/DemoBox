Welcome to Android CodePad.
Android CodePad is a simple source code viewer that supports syntax highlighting. it supports all C-like, Bash-like, and XML-like languages. it automatically selects the syntax depending on the file.

Android CodePad uses WebKit and google-code-prettify to do the syntax highlighting.

Here's some examples:.

import string, sys

# If no arguments were given, print a helpful message
if len(sys.argv)==1:

    print 'Usage: celsius temp1 temp2 ...'
    sys.exit(0)
	
# Loop over the arguments
for i in sys.argv[1:]:
    try: 
        fahrenheit=float(string.atoi(i))
    except string.atoi_error:
	print repr(i), "not a numeric value"
    else:
	celsius=(fahrenheit-32)*5.0/9.0
	print '%i\260F = %i\260C' % (int(fahrenheit), int(celsius+.5))
